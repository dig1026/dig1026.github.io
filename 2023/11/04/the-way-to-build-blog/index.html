the way to build blog

参照csdn中的方法搭建了博客，中途很多都卡住了，整理一遍思路

[TOC]

一.搭建

下载node

下载git

安装完后通过cmd（win+r再输入cmd)检查是否安装成功

​ 附：Node版本命令：node -v

​ Git版本命令：git –version

​ 以上是成功安装的示例

​ (如果出现安装错误需要考虑系统文件是否有缺失，最直接的方法是重装系统（）一般不推荐，因为使用的是新电脑没有重要文件就换硬盘顺便重装了)

二.注册github账号创建仓库

注册账号github

​ 点击右上角sign in开始注册

​ （成功后邮箱会收到信件）

创建仓库

​ 点击左边绿色键“creaye repository”

​ 注意选择public

​ 格式为：用户名.github.io（推荐写注册时的用户名）

​ 生成仓库地址，推荐使用ssh

三.安装hexo

​ 磁盘中创建一个用于存放的文件夹（名字自定义，为方便之后寻找可写blog)

​ 选中文件夹右击打开git bush here

​ 输入命令：npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 输入命令：cnpm install -g hexo-cli

​ 输入命令：hexo init

​ （成功后文件夹中会出现很多文件）

​ 输入命令：hexo s

​ 在浏览器中输入localhost:4000可浏览到博客

​ ctrl+c关闭

四.设置ssh

​ git中输入命令：cd~/.ssh

​ 输入命令：ssh-keygen -t rsa -C’注册时的邮箱地址’

​ 去c盘中（C：\用户\用户名目录）用记事本打开id_rsa.put文件（不要双击——）

​ 里面的内容全部复制

​ 在GitHub主页头像下点击settings（设置）

​ 点击SSH and GDG keys，点击New SSH key（右边第一个绿色键）

​ 将复制的内容输入下格里面，名字可随意取

​ 验证：

​ 输入命令：ssh -T git@github.com并输入yes，行末会显示出用户名（邮箱也会有提醒）

​ 绑定用户名和邮箱

​ 输入命令:git config –global user.name “注册时的用户名”

​ 输入命令:git config –global user.email “注册时的邮箱”

五.上传博客

​ 找到之前生成的仓库地址（可点击主页左边的仓库复制，也许是因为没创建好博客所以还是生成时的界面（？ ）

​ 打开文件夹下的_config.yml

​ 往下滑到最后改为以下格式（注意都是要严格按照缩进的）

deploy:
    type: git
    repo: 复制的网址
    branch: main

​ 输入命令：cnpm install hexo-deployer-git安装上传工具

新建文章：（什么都不编辑会有一个hello world，可以先不写）

​ 输入命令：hexo new”文章名称”会给出文章所在位置，点击即可使用编辑器编辑（需下载编辑器，可用typora）

​ 输入命令：hexo s浏览

​ 输入命令：hexo g生成文件

​ 输入命令：heso d部署到github

GitHub中此时多了很多文件，浏览器中输入：用户名.github.io即可浏览（未更新需要等待一段时间）
